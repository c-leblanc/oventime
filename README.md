OvenTime

Aperçu en temps réel du réseau électrique en France pour savoir s’il est plutôt “vert” (énergie bas-carbone disponible) ou s’il risque d’utiliser du gaz. Donne un simple indicateur pour décider si c’est le bon moment de brancher un appareil.

Bot Telegram accessible à t.me/oventime_bot

Structure

    src/oven_time: logique principale (bot, API, traitement des données)

    requirements.txt: dépendances

    run_bot.py / run_api.py: scripts d’entrée
